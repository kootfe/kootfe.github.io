Allow

Allow path positions

Restricts units to move only through the specified positions.

Parameter Value Explanation Note
POS_LIST x1,y1_x2,y2 Example: 0,4_1,2 means one allowed tile at (0,4) and another at (1,2). Coordinates start at 0x0 in the top-left corner.

Allow click_on_tile

Controls whether certain tiles can be clicked.

Parameter Value Explanation Note
all - Allows clicking every tile. -
none - Blocks clicking on all tiles. -
position x,y Allows clicking only on the tile at that specific coordinate. 0x0 is the top-left tile.

Allow path

Enables or blocks all unit movement paths.

value explanation
all Units can move on any valid path.
none Units cannot move anywhere.

Dialogue

Dialogue continue

Moves forward to the next part of the dialogue.

Disable

Disable

Turns off specific UI elements or interactions.

value explanation
end_turn_button Disables the “End Turn” button.
map_interactions Blocks all interactions with the map.
start_turn_popup Hides the start-of-turn popup.
cancel_on_transport Disables the cancel button shown when selecting a transport unit.
